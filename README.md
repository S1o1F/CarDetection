# Детекција и броење на возила во видео

Демо апликација за детекција и броење на возила во видео користејќи YOLOv8 и OpenCV.

## Опис

Оваа апликација користи YOLOv8 модел за детекција на возила (автомобили, мотори, автобуси, камиони) во видео и ги брои колку возила поминале низ одредена линија во видеото.

## Инсталација

1. Инсталирајте ги потребните библиотеки:

```bash
pip install -r requirements.txt
```

2. YOLOv8 моделот ќе се симне автоматски при првото извршување.

## Користење

### Основна употреба

```bash
# Ако видеото е во videos папката
python vehicle_detection.py --video videos/input_video.mp4

# Или со релативна патека
python vehicle_detection.py --video videos\input_video.mp4
```

```bash

# Со зачувување на излезното видео
python vehicle_detection.py --video videos/input_video.mp4 --output output.mp4

# Со прилагодена линија за броење (60% од висината)
python vehicle_detection.py --video videos/input_video.mp4 --line 0.6

# Со повисок confidence threshold
python vehicle_detection.py --video videos/input_video.mp4 --confidence 0.7

# Без прикажување на видеото
python vehicle_detection.py --video videos/input_video.mp4 --no-display
```


## Како работи

1. **Детекција**: Користи YOLOv8 модел за детекција на возила во секој фрејм
2. **Следење**: Следи позиција на детектираните возила
3. **Броење**: Брои возила кога поминуваат низ одредена линија
4. **Визуелизација**: Прикажува детекции, линија за броење и бројачи


## Резултати

По обработката, апликацијата прикажува:
- Вкупно број на возила
- Број на возила што поминале надолу
- Број на возила што поминале нагоре
